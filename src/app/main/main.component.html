Welcome to the Knora User Interface Research Layer | KUIRL

This is a simple app to work with Knora, a software framework for storing, sharing, and working with primary sources and data in the humanities.

[DEVELOPER REMARK N° 1]

This is the landing page for not registered or logged-in users. It will show the following elements:

  1. a fulltext search panel
  2. a short description about the app and Knora with an information about the end-user (for who is the software?). We will ask a question about the kind of data, a user is looking for and we offer links to FORS in case of "quantitative date", to NIE-INE in case of "edition data". In other case (qualitative data) the user can start with his own project here.
  3. list of projects and info box about them
  4. get more info, link to (workflow) manual / documentation
  5. info about the DaSCH and the partners
  6. footnote
